Cats & Coding

Welcome to Cats & Coding!

This site is dedicated to all cat lovers who also enjoy coding. Below, you'll find a simple R script using the tidyverse package to analyze cat data.

Example Tidyverse R Code

# Load the tidyverse package
library(tidyverse)

# Create a sample dataset of cats
cats <- tibble(
    name = c("Whiskers", "Mittens", "Shadow"),
    age = c(2, 5, 3),
    weight = c(4.5, 6.2, 5.1)
)

# Summarize the average age and weight of the cats
summary <- cats %>%
    summarise(
        avg_age = mean(age),
        avg_weight = mean(weight)
    )

# Print the summary
print(summary)
